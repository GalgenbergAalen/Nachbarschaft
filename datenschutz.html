2. Zweck dieser Website

Diese Website ist eine privat organisierte Nachbarschaftsseite. Sie dient dazu, Informationen, Termine und Hinweise für Anwohnerinnen und Anwohner bereitzustellen.

3. Verarbeitung beim Besuch der Website

Beim Aufruf dieser Website werden technisch erforderliche Daten verarbeitet, damit die Seite korrekt angezeigt und sicher betrieben werden kann.

Dazu können insbesondere gehören: IP-Adresse, Datum und Uhrzeit des Zugriffs, aufgerufene Datei bzw. Seite, Browsertyp und Browserversion, Betriebssystem, Referrer-URL und Internetprovider.

8. Kommentarfunktion über Giscus / GitHub Discussions

Auf der Seite „Austausch“ ist eine Kommentarfunktion über den Dienst Giscus eingebunden. Giscus nutzt GitHub Discussions, um Kommentare bereitzustellen.

Bereits beim Laden der Kommentarfunktion kann eine Verbindung zu Servern von Giscus und GitHub hergestellt werden. Dabei können insbesondere technische Daten wie IP-Adresse, Browserinformationen sowie Angaben zum Aufruf der Seite verarbeitet werden.

Wenn du selbst kommentieren möchtest, ist in der Regel eine Anmeldung bei GitHub erforderlich. Die weitere Verarbeitung personenbezogener Daten erfolgt dann auch durch GitHub im Rahmen der Nutzung von GitHub-Diensten.

Die Einbindung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO. Das berechtigte Interesse besteht darin, eine einfache Möglichkeit für nachbarschaftlichen Austausch und Diskussionen bereitzustellen.

9. Reichweitenmessung mit GoatCounter

Diese Website verwendet GoatCounter zur datensparsamen statistischen Auswertung der Nutzung. Nach Angaben des Anbieters arbeitet GoatCounter ohne Cookies, ohne Speicherung von IP-Adressen und ohne dauerhafte Kennungen im Browser.

Erfasst werden nach Angaben des Anbieters ausschließlich zusammengefasste statistische Informationen, etwa über Seitenaufrufe und Klicks auf ausgewählte Funktionen dieser Website, zum Beispiel Veranstaltungs- und Kalenderlinks.

Die Nutzung von GoatCounter erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO. Das berechtigte Interesse besteht in der bedarfsgerechten, technischen und inhaltlichen Verbesserung dieser Website.

Diese Website wird über GitHub Pages bereitgestellt. Außerdem werden externe Dienste von GitHub im Zusammenhang mit Giscus/GitHub Discussions sowie GoatCounter eingebunden.
