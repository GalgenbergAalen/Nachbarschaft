Beim Aufruf dieser Funktion können Daten an Giscus und GitHub übertragen werden, insbesondere technische Verbindungsdaten wie IP-Adresse, Browserinformationen und weitere Nutzungsdaten.

Wenn du die Kommentarfunktion nutzt oder dich bei GitHub anmeldest, werden weitere Daten entsprechend der Verarbeitung durch GitHub verarbeitet. Auf diese Verarbeitung durch GitHub habe ich keinen vollständigen Einfluss.
